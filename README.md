## Portfolio Projects (JavaScript)
A collection of interactive, responsive, and functional web applications built using vanilla JavaScript, HTML5, and CSS3. This repository showcases fundamental frontend development concepts, DOM manipulation, asynchronous API handling, and state management.
## 🚀 Projects Included
The repository contains the following individual projects:
## 1. Currency Converter (/currency-converter)>

* Description: A real-time currency conversion tool.
* Key Features: Dynamic exchange rate fetching, multi-currency support, and instant calculation updates.

## 2. Expense Tracker (/expense-tracker)>

* Description: A personal finance application to monitor income and expenses.
* Key Features: Transaction history logs, balance tracking, and category-based sorting.

## 3. Quiz Game (/quiz-game)>

* Description: An interactive trivia game to test user knowledge.
* Key Features: Multiple-choice questions, live score tracking, timer integration, and a final summary dashboard.

## 4. Recipe Finder (/recipe-finder)>

* Description: A search engine for cooking recipes.
* Key Features: Ingredient-based search queries, detailed cooking instructions, and image previews.

------------------------------
## 🛠️ Tech Stack

* HTML5: Structured semantic layout.
* CSS3: Custom styling, flexbox/grid alignment, and responsive design.
* JavaScript (ES6+): Event listeners, DOM updates, and API integration.

## 💻 Getting Started
Follow these steps to run any of the projects locally on your machine.
## Prerequisites
You only need a modern web browser (e.g., Chrome, Firefox, Edge, Safari) and a code editor like VS Code.
## Installation

   1. Clone the repository:
   
   git clone https://github.com/aaronkipkirui/portfolio-projects-js
   
   2. Navigate to the project directory:
   
   cd portfolio-projects-js
   
   3. Open a project:
   * Navigate into any specific project folder (e.g., cd quiz-game).
      * Open the index.html file directly in your browser, or use a VS Code extension like Live Server to host it locally.
