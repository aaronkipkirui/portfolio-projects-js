# Currency Converter

A lightweight, responsive web application built with vanilla **HTML**, **CSS**, and **JavaScript** that allows users to seamlessly convert values between different global currencies. This project is part of my JavaScript portfolio.

## 🚀 Features

* **Real-Time Exchange Rates:** Dynamically fetches up-to-date conversion rates using a live currency API.
* **Responsive Design:** Optimized user experience across mobile, tablet, and desktop viewports.
* **Interactive UI:** Dynamic dropdown menus for easy currency selection and intuitive layout.
* **Instant Conversions:** Quick value calculations with zero page reloads.

## 🛠️ Technologies Used

* **HTML5:** Structures the layout and user inputs.
* **CSS3:** Custom styles, visual layout, and responsive media queries.
* **JavaScript (ES6):** Handles API requests, currency manipulation logic, and UI updates.
* **Fetch API:** Handles network requests to pull third-party exchange rates asynchronously.

## ⚙️ How to Run Locally

You do not need to install any heavy packages or build configurations to run this app. Simply follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aaronkipkirui/portfolio-projects-js/new/main/currency-converter
   ```
2. **Navigate to the project directory:**
   ```bash
   cd portfolio-projects-js/currency-converter
   ```
3. **Open the application:**
   * Double-click the `index.html` file to view it directly in your browser.
   * Alternatively, serve it via an extension like VS Code's **Live Server** for instant reloads.

## 📖 Usage

1. Enter the total amount you want to convert in the input field.
2. Select your origin currency from the **"From"** dropdown menu.
3. Select your target currency from the **"To"** dropdown menu.
4. Click the conversion button to instantly calculate the updated values.
