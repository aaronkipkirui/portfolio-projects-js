## 💰 Vanilla JS Expense Tracker
A lightweight, responsive web application designed to help users log income, track expenses, and view their real-time financial balance. Built using pure vanilla web technologies as part of a JavaScript portfolio collection.
## 🚀 Features

* Real-Time Calculation: Automatically updates overall balance, total income, and total expenses.
* Transaction History: Adds and displays a history log of recent financial movements.
* Dynamic Classes: Highlights items dynamically (e.g., green indicators for income, red indicators for expenses).
* Persistent Storage: Uses localStorage to save your transactions so data persists across browser refreshes.
* Fully Responsive: Styled with CSS to provide a clean layout across desktop and mobile devices.

## 📖 How to Use

   1. Add a Transaction: Type the name of the transaction in the description input field (e.g., "Salary" or "Coffee").
   2. Enter Amount: Input the monetary value.
   * Use a positive number for income (e.g., 2500).
      * Use a negative number for expenses (e.g., -15).
   3. Submit: Click the "Add Transaction" button. The history container and dashboard metrics will update automatically.
   4. Delete an Entry: Click the inline delete button x next to any history entry to remove it permanently.
